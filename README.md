## 🚀 Highlighted Projects

### 📈 1. [Financial Market Intelligence Agent](https://github.com/abdulsalamfaiz2024-cmd)
*An end-to-end agentic AI platform that performs real-time financial market analysis and technical forecasting.*
*   **Architecture:** Multi-agent system (Research, Technical Analysis, Sentiment, Prediction, and Reporting agents) orchestrated with Google ADK.
*   **Pipeline:** Incorporates RAG (Retrieval-Augmented Generation) using ChromaDB and real-time financial stream feeds (Yahoo Finance / Alpha Vantage API).
*   **UI & API:** Powered by a fast FastAPI backend and an interactive Streamlit frontend with Plotly visualizations.
*   **Technologies:** `Python`, `Google Gemini API`, `FastAPI`, `Streamlit`, `ChromaDB`, `scikit-learn`, `Docker`, `AWS ECS`.

### 🧠 2. [PhonixPro — AI-Driven SaaS Data Analysis Platform](https://github.com/abdulsalamfaiz2024-cmd)
*A professional-grade cloud platform designed to automate statistical modeling and reporting for non-technical users.*
*   **Core Logic:** Enables drag-and-drop CSV/Excel parsing, smart type inference, interactive visualizations, and automated PDF export.
*   **AI Integrations:** Features a "Chat with Data" interactive analyst console using LLM APIs to translate questions into insights and one-click smart data cleaning.
*   **Technologies:** `Next.js 16 (React)`, `Tailwind CSS v4`, `Python`, `PostgreSQL`, `Redis`, `Framer Motion`, `Gemini AI`.

### 🗺️ 3. [Yemen Nexus — Real-Time Country Dashboard](https://github.com/abdulsalamfaiz2024-cmd)
*A unified operational dashboard bringing together fragmented public and private datasets across crucial sectors in Yemen.*
*   **Pipeline:** Automated Python ETL pipelines fetch data regularly (economy, health, weather, risks) and aggregate it into a unified DB.
*   **Visualizations:** Highly intuitive Power BI interfaces mapping historical trends, weather models, and humanitarian indicators.
*   **Impact:** Reduced leadership reporting cycles by over 30%, giving decision-makers a single source of truth.
*   **Technologies:** `Python`, `Power BI`, `SQL Server`, `REST APIs`, `Azure`.

### 🗄️ 4. [TNSC Relational Database Architecture](https://github.com/abdulsalamfaiz2024-cmd)
*A production-ready database system migrating consulting workflows from disorganized spreadsheets to a high-integrity PostgreSQL database.*
*   **Complexity:** 36 fully-normalized tables, 25+ indexes, 6 specialized reporting views (staff utilization, grant disbursement, project beneficiaries).
*   **Advanced Features:** Bilingual column setups (`name`/`name_ar`), UUID primary keys, soft-delete triggers, and database-level computed fields.
*   **Technologies:** `PostgreSQL 15+`, `Database Design`, `Python`.
